### Step 2: Enable Installation from Unknown Sources

⚠️ **IMPORTANT**: Android blocks installations from unknown sources by default

**For Android 8.0+:**
```
1. Open Settings app
2. Go to: Apps & notifications
3. Tap "Advanced" or "Permission manager"
4. Look for "Install unknown apps"
5. Find your browser or file manager
6. Toggle "Allow from this source" ON
```

**For Android 6.0-7.x:**
```
1. Open Settings
2. Go to: Security
3. Find "Unknown sources"
4. Toggle ON
5. Confirm when prompted
```

## ❌ TROUBLESHOOTING

### "Installation blocked" Error

**Solution:**
```
1. Go to Settings → Apps
2. Find your file manager
3. Tap "Permissions"
4. Enable "Storage" or "Files"
5. Try install again
```

### "Parse error" When Installing

**Solution:**
```
1. APK file might be corrupted
2. Download again from GitHub
3. Delete old APK file
4. Try installation again

OR

1. Check Android version is 7.0+
2. Phone might not have enough storage
3. Free up 200 MB space
4. Try again
```

### "App crashes on startup"

**Solution:**
```
1. Clear app cache:
   - Settings → Apps → Law Empire
   - Storage → Clear Cache
   
2. Reinstall:
   - Uninstall app
   - Restart phone
   - Reinstall APK
   
3. Check Android version (7.0+ required)

4. Ensure internet connection

5. Report issue on GitHub:
   https://github.com/simeonpangu1/law-empire-sierra-leone/issues
```

### "Not enough storage"

**Solution:**
```
1. Delete unused apps
2. Clear app caches:
   - Settings → Apps → Storage
   - Clear Cache
3. Move photos/videos to cloud
4. Free up at least 200 MB
5. Try install again
```

### "Unknown error"

**Solution:**
```
1. Restart phone (turn off/on)
2. Re-enable "Unknown sources"
3. Try installing again
4. If still fails, try Option 2 or 3
```

## ⚡ PERFORMANCE TIPS

**To run game smoothly:**

```
1. Close other apps:
   - Swipe up from bottom → Close apps
   - Keep only essential apps open

   - Settings → About Phone

   - Settings → Graphics → Set to "Medium" or "Low"

4. Use WiFi not mobile data:
   - Better connection
   - No data usage

5. Keep phone cool:
   - Remove phone case
   - Don't play in direct sunlight
   - Use cooling fan if available
```
