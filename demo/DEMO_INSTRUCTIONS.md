### Local Demo Setup

#### Prerequisites
- Unity 2022 LTS installed
- .NET Framework installed
- Git installed

#### Steps

1. **Clone the repository**
```bash
git clone https://github.com/simeonpangu1/law-empire-sierra-leone.git
cd law-empire-sierra-leone
```

2. **Open in Unity**
```
- Open Unity Hub
- Click "Open Project"
- Select: law-empire-sierra-leone/frontend/unity
- Wait for import (5-10 minutes)
```

3. **Play the Demo**
```
- In Unity Editor, click Play button (top center)
- Or press Ctrl+P (Windows) or Cmd+P (Mac)
```

## Demo Gameplay Walkthrough

### Scene 1: Login Screen
- **Create Account**: Enter email, password, username
- **Skip**: Click "Guest Play" to start immediately
- **Action**: Click "Start Game"

### Scene 2: Character Creation
- **Select Gender**: Male/Female/Other
- **Choose Appearance**: Hair, skin tone, clothing
- **Set Personality**: Pick 3 traits
- **Action**: Click "Create Character"

### Scene 3: Main Game Screen
- **Display**: Your stats (Level, Coins, Experience)
- **Available Actions**:
  - "View Cases" - See available legal cases
  - "Go to Office" - Manage your office
  - "Visit Court" - Enter courtroom
  - "Check Profile" - View your stats

### Scene 4: Case Selection
- **Case 1**: "The Theft at Freetown Market" (Difficulty: Easy)
- **Case 2**: "Boundary Dispute in Bo" (Difficulty: Medium)
- **Case 3**: "Contract Breach" (Difficulty: Medium)

### Scene 5: Courtroom
- **Read Case Details**: Understand the legal issue
- **Investigation Phase**: 
  - Interview witnesses
  - Review evidence
  - Make strategic choices
- **Trial Phase**:
  - Cross-examine opponents
  - Present arguments
  - Deliver closing statement
- **Verdict Phase**:
  - See outcome
  - Receive rewards (coins, experience)
  - Case marked as won/lost
